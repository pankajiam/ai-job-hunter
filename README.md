#  AI Job Hunter Agent

An AI-powered job search platform for students and freshers.

##  Live Demo
👉 https://ai-job-hunter-cmxnveifv9m4mfj9ywdpds.streamlit.app

##  Features
- 📄 Resume Analyzer & ATS Scorer
- 🔍 Job Search
- 🎯 Resume Matcher
- ✉️ Cold Email Generator
- 📊 Application Tracker
- 🎓 AI Career Coach

##  Tech Stack
- Python
- Streamlit
- Groq AI (LLaMA 3.1)
- SQLite
- SMTP

##  Setup Locally
1. Clone the repo
2. Create virtual environment: `python -m venv venv`
3. Activate: `venv\Scripts\activate`
4. Install packages: `pip install -r requirements.txt`
5. Add your GROQ_API_KEY in `.env` file
6. Run: `streamlit run app.py`
